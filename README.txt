1. Introduction

An ELF file symbol obfuscator based on debugedit tool in the rpm package: 

http://www.rpm.org/

2. Set Up and Build
Install Cygwin and related packages in your computer, then run
# make
If you're lucky, you have an exe file named 'obfuscateElfStr.exe'  

3. Usage

#./obfuscateElfStr.exe YourSOFile
